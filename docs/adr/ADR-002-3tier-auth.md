# ADR-002: 3-Tier 인증 설계

| 항목 | 내용 |
|---|---|
| **상태** | 승인됨 |
| **결정일** | 2024-12 |
| **결정자** | 프로젝트 아키텍트 |

---

## 컨텍스트

AI 게이트웨이는 다양한 배포 환경(로컬 개발, CI, 프로덕션)에서 실행되어야 합니다.

**요구사항**

- 외부 DB(SurrealDB)가 없는 환경에서도 단독 실행 가능해야 함
- 멀티테넌시: Key별로 `project_id`, `role`, `rpm` 격리
- JWT 기반 단기 토큰(15분 만료) 지원 — 서비스 간 인증에 활용
- DB 장애 시 Static 키로 Fallback, 서비스 중단 없음

## 인증 흐름

```
요청의 Authorization: Bearer <token>
       │
       ▼
┌─────────────────────────────────────────────────────┐
│ Tier 1: JWT 검증                                     │
│  - HS256 서명, exp 만료, jti 재사용 방지              │
│  - 성공 시 → Principal 반환 (DB 조회 없음)           │
└──────────────────────┬──────────────────────────────┘
                       │ JWT 아님 또는 서명 불일치
                       ▼
┌─────────────────────────────────────────────────────┐
│ Tier 2: SurrealDB API Key 조회                       │
│  - SHA-256 해시로 `api_key:{hash}` 레코드 조회       │
│  - 성공 시 → Principal (project, role, rpm)          │
│  - 5분 메모리 캐시로 DB 부하 완화                     │
└──────────────────────┬──────────────────────────────┘
                       │ DB 없음 또는 조회 실패
                       ▼
┌─────────────────────────────────────────────────────┐
│ Tier 3: 환경변수 Static Fallback                     │
│  - AEGIS_API_KEYS 또는 TOML [api_keys] 섹션 조회    │
│  - 성공 시 → Principal 반환                          │
└──────────────────────┬──────────────────────────────┘
                       │ 3 Tier 모두 실패
                       ▼
              401 Unauthorized
```

## 결정 근거

### DB 의존성 제거 (Tier 3 Static)
게이트웨이가 보안의 핵심 경로에 있기 때문에 **DB 장애가 곧 서비스 완전 중단**으로 이어지는 구조를 피했습니다. 환경변수만으로 기동 가능하면 로컬 개발, CI 환경에서도 별도 인프라 없이 통합 테스트를 실행할 수 있습니다.

### JWT 단기 토큰 (Tier 1)
서비스 간 통신(예: SliceRAG → AegisLLM)에서 장기 API Key 노출 없이 15분 만료 JWT를 발행합니다. `jti` 클레임으로 토큰 재사용 공격을 방지합니다.

### 메모리 캐시 (Tier 2)
SurrealDB 조회 결과를 5분간 캐시하여 요청마다 DB 왕복을 방지합니다. 캐시 만료 후 자동 갱신되므로 Key 폐기 시 최대 5분 내 반영됩니다.

## 트레이드오프 수용

- Static Key는 환경변수 유출 시 위험 → 프로덕션에서는 SurrealDB Tier로 전환하고 환경변수 비우기를 권장
- 3-Tier 순서로 인해 Tier 1이 빠르고 Tier 3이 가장 느림 → Tier 3은 키 수가 적어 선형 탐색 비용 무시 가능

## 관련 코드

- `src/auth.rs`: `authenticate()`, `RateLimiter`, `Principal`
- `src/config.rs`: `PrincipalConfig`, `JwtConfig`, `DatabaseConfig`
